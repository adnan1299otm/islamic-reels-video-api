\# Islamic Instagram Reels - Video Processing API



FFmpeg-based video processing API for creating Islamic Instagram Reels with Hadith overlays.



\## Features



\- ✅ Remove original video audio

\- ✅ Add Islamic nasheed/music

\- ✅ Add text overlays (Hadith + Source)

\- ✅ Resize to 9:16 vertical (1080x1920)

\- ✅ Export Instagram-compatible MP4



\## Tech Stack



\- Flask (Python)

\- FFmpeg

\- Deployed on Render.com
